# Walking Infinity Nuke

This mod makes the "Infinity Nuke" from [Industrial Foregoing](https://www.curseforge.com/minecraft/mc-mods/industrial-foregoing) walk around as a mob.
Beware! It attracts lightning, gets angry when hit, and can even multiply.

Industrial Foregoingの「Infinity Nuke」が、歩き回るモブとして出現するようになるModです。
雷を引き寄せたり、殴ると怒って起爆したり、ネザースターで増殖したりします。
